# Tutorials
To explain how to use ZEN-norm, runnable examples are included within Jupyter and R markdown notebooks.

<a id="zen-tutorials"></a>
## ZEN Tutorials
This folder contains:
* [ZEN Tutorial](https://github.com/Genome-Function-Initiative-Oxford/Zone-Equalisation-Normalisation/blob/main/tutorials/zen_tutorials/ZEN_Tutorial.ipynb): Explains how to use module `ZoneNorm` to create ZEN normalised bigWigs
* [ReverseNorm Tutorial](https://github.com/Genome-Function-Initiative-Oxford/Zone-Equalisation-Normalisation/blob/main/tutorials/zen_tutorials/ReverseNorm_Tutorial.ipynb): Explains how to use module `ReverseNorm` to create unnormalised bigWigs from pre-normalised bigWigs
* [CompareNorm Tutorial](https://github.com/Genome-Function-Initiative-Oxford/Zone-Equalisation-Normalisation/blob/main/tutorials/zen_tutorials/CompareNorm_Tutorial.ipynb): Explains how to use module `CompareNorm` to compare and quantify genome-wide alignment between normalisation methods via Wasserstein distance

<a id="diffbind"></a>
## DiffBind
This folder contains:
* [DiffBind Analysis](https://github.com/Genome-Function-Initiative-Oxford/Zone-Equalisation-Normalisation/blob/main/tutorials/diffbind/DiffBind_Analysis.Rmd): Code used to calculate scaling factors for LIB, RLE and TMM normalisation, as well as run differential accessibility analysis with ZEN normalisation
